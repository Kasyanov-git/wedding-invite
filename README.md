# Свадебное приглашение — премиальное веб-приложение

Элегантный одностраничный лендинг для приглашения гостей на свадьбу: с полноэкранной открыткой, анимацией «открытия конверта», видео-фоном из развевающегося тюля и формой для сбора ответов гостей.

## Возможности

- **Полноэкранная открытка** — фото молодожёнов на весь экран с наложенным текстом приглашения.
- **Стартовый экран конверта** — одно полноэкранное видео-интро, которое автоматически проигрывается при открытии сайта. В конце видео срабатывает CSS-вспышка и появляется основное приглашение. Если видео не загрузится — автоматически включается CSS-анимация конверта.
- **Видео-фон** — закольцованное видео с тюлем в нежных тонах + CSS-fallback, если видео недоступно.
- **Блочная структура** — каждый блок вынесен в отдельный `<section>` и легко редактируется. Каждый блок имеет место для PNG/SVG-иллюстрации чёрной кистью.
- **Прозрачные блоки** — без подложек, текст чёрный, лёгкий `text-shadow` для читаемости на видео-фоне.
- **Адаптивность** — мобильная вёрстка с подстройкой под планшеты и десктопы.
- **Форма RSVP** — имя, подтверждение участия, **вопрос про второй день**, количество гостей (выбор от 1 до 5), **имена всех гостей** (для рассадки), напитки.
- **Хранилище** — SQLite база данных.
- **Выгрузка данных** — админка с формой входа, выгрузка только в Excel (`.xlsx`).

## Быстрый старт

### 1. Установка зависимостей

```bash
cd wedding-invite
python3 -m venv .venv
source .venv/bin/activate   # Windows: .venv\Scripts\activate
pip install -r requirements.txt
```

### 2. Запуск локально

```bash
python app.py
```

Откройте в браузере: `http://localhost:5000`

### 3. Вход в админку

Перейдите по адресу: `http://localhost:5000/admin` или `http://localhost:5000/login`

- Введите пароль: `wedding2025` (смените через переменную окружения `ADMIN_PASSWORD`)
- Нажмите «Войти»

В админке доступны:
- статистика по ответам,
- список всех гостей,
- кнопка выгрузки в Excel.

### 4. Переменные окружения

| Переменная | Описание | По умолчанию |
|------------|----------|--------------|
| `ADMIN_PASSWORD` | Пароль для админки | `wedding2025` |
| `SECRET_KEY` | Секретный ключ Flask | `wedding-invite-secret-key-change-me` |
| `DATABASE_PATH` | Путь к SQLite | `wedding.db` |
| `HOST` | Хост для запуска | `0.0.0.0` |
| `PORT` | Порт | `5000` |
| `FLASK_DEBUG` | Режим отладки | `False` |

Пример запуска с кастомным паролем:

```bash
ADMIN_PASSWORD=supersecret SECRET_KEY=your-secret-key python app.py
```

## Как настроить под свою свадьбу

Отредактируйте `static/index.html`:

1. **Имена и дата** — замените `Александр` и `Екатерина`, а также `15 августа 2025` в блоках `hero-section`, `date-section`, `closing-section`.
2. **Фотография** — положите фото молодожёнов в `static/images/couple.jpg`. Для лучшего эффекта используйте вертикальное или квадратное фото с мягким светом.
3. **Иллюстрации** — для каждого блока в `static/images/` лежит файл `illustration-*.png` (или `.svg`). Замените их на свои рисунки чёрной кистью с прозрачным фоном. Если файла нет — блок просто покажется без иллюстрации.
4. **Видео-фон** — смотрите раздел «Видео-фон» ниже.
5. **Место** — в `location-section` укажите название ресторана и адрес, обновите ссылку на Яндекс Карты.
6. **Таймлайн** — измените время и описание этапов в `timeline-section`.
7. **Второй день** — отредактируйте текст перед вопросом про второй день в `rsvp-section` (дата, описание завтрака).
8. **Пожелания** — отредактируйте текст в `wishes-section`.
9. **Дата свадьбы для обратного отсчёта** — в `static/js/app.js` измените строку `const weddingDate = new Date('2025-08-15T15:00:00').getTime();`.

## Шрифты

Шрифты загружаются локально из папки `static/fonts/` (Google Fonts больше не используется).

Нужно **3 файла**:

- **Marggraff Kursiv Zarte.ttf** — курсивный декоративный шрифт для имён молодожёнов, заголовков блоков и цифр.
- **Vetrino.otf** — мелкий текст, метки формы, кнопки, подсказки.
- **Akademie Alte.ttf** — подзаголовки, пожелания гостям, адрес, дата, пояснения к форме.

Подробнее — в `static/fonts/README.md`. Если хотите использовать другой шрифт, замените соответствующий файл в `static/fonts/` (сохранив имя) или измените пути и `font-family` в `@font-face` в `static/css/styles.css`.

## Видео на стартовом экране (интро)

Для стартового экрана нужно одно видео:

**`static/video/envelope-intro.mp4`** — видео-интро, которое автоматически проигрывается при открытии сайта. В нём должен быть показан весь сюжет: лежащий конверт, его открывание, и в конце — момент для перехода к приглашению. После завершения видео срабатывает CSS-вспышка и появляется основное приглашение.

Рекомендации:
- **Разрешение:** 1080p или 720p, на весь экран (`object-fit: cover`).
- **Длительность:** 5–15 секунд.
- **Кодек:** H.264 (MP4), без звука.
- **Размер:** желательно не более 5–10 Мб после сжатия.
- **Фон:** кремовый/шампань/пудровый, чтобы гармонировал с основным дизайном.
- **Конец видео:** в финале должен быть кадр, после которого логично показать приглашение (например, конверт полностью открыт, листок вынут, или просто плавный исчезающий конверт).

Сожмите видео скриптом `optimize_video.py`:

```bash
python optimize_video.py путь/к/envelope-intro.mp4
# Скопируйте результат в static/video/envelope-intro.mp4
```

Если видео не загрузится или не поддерживается браузером, автоматически включится CSS-анимация конверта (старый вариант).

## Видео-фон

### Подготовка видео

1. Сгенерируйте видео с развевающимся тюлем в нежных кремово-пудровых тонах.
2. Рекомендуемые параметры:
   - **Разрешение:** 720p (1280x720) достаточно для фона.
   - **Длительность:** 10–20 секунд (видео будет зациклено).
   - **Кодек:** H.264 (MP4) для совместимости со всеми браузерами.
   - **Размер:** желательно не более 5–8 Мб после сжатия.
3. Положите исходный файл в папку проекта.

### Сжатие видео

Если установлен ffmpeg:

```bash
# Ubuntu/Debian
sudo apt install ffmpeg

# macOS
brew install ffmpeg
```

Запустите скрипт:

```bash
python optimize_video.py /путь/к/вашему_видео.mp4
```

Скрипт создаст оптимизированный файл `static/video/tulle.mp4`.

Если ffmpeg не установлен, можно сжать вручную:

```bash
ffmpeg -i input.mp4 -vf "scale='min(1280,iw)':-2" -c:v libx264 -crf 24 -preset slow -an -movflags faststart -pix_fmt yuv420p static/video/tulle.mp4
```

### Проверка

После добавления видео откройте сайт. Если видео не загрузится или устройство его не поддерживает, автоматически включится CSS-анимация фона.

## Деплой на сервер

### Вариант 1: Простой VPS (Ubuntu)

```bash
# Клонируйте/загрузите проект на сервер
cd /var/www/wedding-invite
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

# Запуск через Gunicorn
pip install gunicorn
gunicorn -w 2 -b 0.0.0.0:8000 app:app
```

Для работы 24/7 настройте `systemd` или `supervisor`.

### Вариант 2: PythonAnywhere / Railway / Render

- Загрузите файлы через Git или веб-интерфейс.
- Укажите `app:app` как точку входа (WSGI).
- Установите зависимости из `requirements.txt`.

### Рекомендации по безопасности

- Обязательно смените `ADMIN_PASSWORD` и `SECRET_KEY` перед публикацией.
- Для продакшена используйте HTTPS.
- Для публичной формы можно добавить защиту от спама (reCAPTCHA, лимит запросов).

## Выгрузка ответов

После входа в админку (`/admin`):

- **Статистика** — количество ответов, придут / не придут / возможно, ожидается гостей.
- **Список гостей** — таблица со всеми данными, включая имена дополнительных гостей для рассадки и ответ про второй день.
- **Excel** — `/admin/export` (единственная кнопка выгрузки).

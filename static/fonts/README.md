# Шрифты приглашения

Положите сюда TTF/OTF-файлы шрифтов, которые хотите использовать. CSS уже настроен на загрузку из этой папки.

## Нужные файлы (3 шрифта)

| Файл | Назначение | Где используется |
|------|------------|------------------|
| `Marggraff Kursiv Zarte.ttf` | Курсивный декоративный | Имена молодожёнов на главном экране и в конце, заголовки блоков, цифры (таймлайн, календарь), название места |
| `Vetrino.otf` | Мелкий текст | Подписи, метки формы, кнопки, подсказки, основной мелкий текст |
| `Akademie Alte.ttf` | Подзаголовки и пожелания | Вводный текст на главном экране, пожелания гостям, адрес, дата, пояснения к форме |

## Как заменить шрифт на другой

1. Скачайте нужный TTF/OTF-файл.
2. Переименуйте его в соответствующее имя из таблицы выше и положите в `static/fonts/`, либо измените путь в `@font-face` в `static/css/styles.css`.
3. Если меняете семейство, поправьте `font-family` в `@font-face` и в CSS-переменных в `:root` (`--font-script`, `--font-serif`, `--font-sans`, `--font-subtitle`).

## Важно

Все три шрифта загружаются в одном начертании (400). Поэтому в CSS для кнопок, заголовков и акцентов сброшены `font-weight` на 400 — браузер не будет пытаться синтезировать жирность, и шрифты отобразятся в оригинальном виде.
